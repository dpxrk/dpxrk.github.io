Work

GitHub || Live Site === KappaBnB

Above is the splash page to a project that I am very proud of, KappaBnb. It is a spin-off to the popular web/phone application, AirBnB. It was accomplished the course of two weeks, and the spin off consists of major bookings to be the capitol building of each state. If you are familiar with the meme, Kappa and "Capi-tol" sounds very similar which is the origination to the name. This project was developed using the technologies React.js, React-Redux, JavaScript and CSS3 on the front end, while using Python, Flask-SQLAlchemy, and Postgres for the backend.

GitHub|| Live Site === InstaLock

Above is the splash to a project that I developed with three other developers: Kevin Pitzer, Robert Vogtritter, and Keith Taylor. It is a spin-off to the popular application, Instagram. We took the twist to have each user be a champion from League of Legends, developed by Riot Games. This project was very enjoyable in the sense that, as a gamer, I was able to incorporate one of my favorite games of all time as the theme of it. This project was developed using the technologies React.js, React-Redux, JavaScript and CSS3 on the front end, while using Python, Flask-SQLAlchemy, and Postgres for the backend.

GitHub || Live Site === Sodium

Sodium was the first project that I have accomplished in my career as a software developer. I had an opportunity to work with three other developers, Justin Payne, Rajiv Hudek, and Steve Guttbinder. This project incorporated using Pug.js, and CSS3 on the front-end, while using Express and Postgres for the backend. The theme of the project oriented around articles which contained "salty" articles (another meme) and was a spin-off to medium.com.
